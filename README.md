# Neovim Config

Uses [LazyVim](https://github.com/LazyVim/LazyVim) distrobution as a starter config.

### Theme:
- [onedark](https://github.com/navarasu/onedark.nvim)

### LazyExtras:
- [prettier](https://www.lazyvim.org/extras/formatting/prettier)
- [typescript](https://www.lazyvim.org/extras/lang/typescript)
- [eslint](https://www.lazyvim.org/extras/linting/eslint)
- [leap](https://www.lazyvim.org/extras/editor/leap)
- [mini-animate](https://www.lazyvim.org/extras/ui/mini-animate)
- [tailwind](https://www.lazyvim.org/extras/lang/tailwind)
- [json](https://www.lazyvim.org/extras/lang/json)
- [python](https://www.lazyvim.org/extras/lang/python)
- [black](https://www.lazyvim.org/extras/formatting/black)


### Additional Plugins:
- [nvim-colorizer](https://github.com/norcalli/nvim-colorizer.lua) - adds highlighting to hex colors for inline color previews.
- [emmet-vim](https://github.com/mattn/emmet-vim) - adds emmet HTML snippets.
- [vim-fugitive](https://github.com/tpope/vim-fugitive) - adds multiple git commands and functionality.
- [gitsigns](https://github.com/lewis6991/gitsigns.nvim) - adds edit/removed line highlighting for git changes.
- [git-blame](https://github.com/f-person/git-blame.nvim) - adds git blame functionality within files.
- [guess-indent](https://github.com/NMAC427/guess-indent.nvim) - automatically sets indentation settings for files based on the current indentations of a file.
- [treesj](https://github.com/Wansmer/treesj) - adds ability to expand/collapse objects for easier viewing and editing.
- [yanky](https://github.com/gbprod/yanky.nvim) - adds additional "yank" functionality such as clippboard history.
  - Installed separately due to errors with LazyExtra's version. Not using SQL database.

### Config Changes:
- indentscope - sped up animation.
- mini-animate - sped up animations.
- tab settings - set to 4 spaces per tab an to use tab character instead of spaces.
